# Vastu Vidya & Design Correction Module Backend

This is the backend for the Vastu Vidya & Design Correction module of the Integrated Home Design and Planning Management System.

## Overview

This module evaluates generated floor plans against Vastu Vidya principles and suggests corrections.

## Tech Stack

- **Language:** Python 3.x
- **Framework:** FastAPI
- **Server:** Uvicorn

## Setup

1.  Create a virtual environment:
    ```bash
    python -m venv venv
    ```
2.  Activate the virtual environment:
    - Windows: `venv\Scripts\activate`
    - Unix/MacOS: `source venv/bin/activate`
3.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
4.  Run the server:
    ```bash
    uvicorn app.main:app --reload
    ```
